# Wanzt Store Bot

## Overview
Bot WhatsApp otomatis untuk toko online yang dirancang khusus untuk menangani pesanan secara otomatis tanpa admin.

## Features
- Otomatisasi Order
- Integrasi Payment Gateway
- Push Notifications
- Dashboard Monitoring

## Tech Stack
- Node.js
- Baileys
- MongoDB
- Express.js
