# Bot WhatsApp Multi-Fitur

## Overview
Bot serbaguna yang bisa digunakan di grup maupun private chat, terintegrasi dengan berbagai API termasuk OpenAI.

## Features
- AI Chat (ChatGPT)
- Auto Downloader (Tiktok, IG, dll)
- Mini Games
- Group Management

## Tech Stack
- Node.js
- Baileys
- OpenAI API
- MySQL
