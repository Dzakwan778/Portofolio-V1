# Aplikasi Pencatatan Keuangan

## Overview
Aplikasi mobile native untuk UMKM agar dapat mencatat pengeluaran dan pemasukan dengan mudah.

## Features
- Sinkronisasi Cloud dengan Firebase
- Mode Offline dengan Room DB
- Laporan Grafik Keuangan
- UI Modern Material 3

## Tech Stack
- Kotlin
- Firebase
- Room DB
- Material 3
